Semi-Automatic Classification Plugin’s User Manual¶

The Semi-Automatic Classification Plugin is a free plugin for QGIS (open source) that allows for the semi-automatic supervised classification of remote sensing images, providing tools to expedite the creation of ROIs, the pre processing phases (image clipping, Landsat conversion to reflectance), the classification process, and the post processing phases (accuracy assessment, land cover change).

Written by Luca Congedo for the Adapting to Climate Change in Coastal Dar es Salaam Project. This plugin requires the installation of SAGA GIS, Numpy, SciPy and Matplotlib. The plugin can be downloaded from the QGIS Plugin Repository.
For more information and tutorials visit the official blog From GIS to Remote Sensing.
